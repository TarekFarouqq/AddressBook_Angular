# Logo Image Instructions

## How to Add Your Logo

1. **Place your logo image** in this folder (`src/assets/images/`)
2. **Name it** `logo.png` (or update the path in the sidebar component)
3. **Recommended size**: 200x200 pixels or similar square dimensions
4. **Supported formats**: PNG, JPG, SVG, WebP

## Current Usage

The logo is displayed in:
- **Sidebar header** (60x60px container with rounded corners)
- **Mobile header** (same logo)

## Tips

- Use a **transparent background** (PNG) for best results
- Keep the logo **simple and clear** at small sizes
- **Square aspect ratio** works best (1:1)
- The logo will be automatically scaled to fit the container
